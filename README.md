# testcontainers-embedded-vertica-demo
This is an example of writing VerticaDB integration test and unit test code using `testcontainers`.

here are two examples: creating a Docker container directly using `GenericContainer` and running a test container by creating a configuration file with `docker-compose`.


> First, you need to configure the docker environment on your local computer!
